# 🎓 Student Management System

A web-based **Student Management System** developed using **ASP.NET Web Forms**, **C#**, **ADO.NET**, and **SQL Server**. The system allows administrators to manage student and teacher records efficiently through a secure admin panel.

---

## ✨ Features

- 🔐 Admin Login & Sign Up
- 👨‍🎓 Student Management (Create, Read, Update, Delete)
- 👨‍🏫 Teacher Management (Create, Read, Update, Delete)
- 🗄️ SQL Server Database Integration
- 📊 User-friendly Admin Dashboard
- ✅ Form Validation

---

## 🛠️ Technologies Used

- ASP.NET Web Forms
- C#
- ADO.NET
- SQL Server
- HTML
- CSS
- Visual Studio

---

## 🗄️ Database

- **Database Name:** SMS
- **Database:** SQL Server

## 🚀 Getting Started

### Prerequisites

- Visual Studio
- SQL Server
- .NET Framework

### Installation

1. Clone this repository.

```bash
git clone https://github.com/YourUsername/StudentManagementSystem.git
```

2. Open the solution in Visual Studio.

3. Attach the **SMS** database in SQL Server.

4. Update the connection string in **Web.config** if required.

5. Build and run the project.
